Программа RAG Pro помогает быстро находить информацию в ваших документах. Вы кладете файлы с договорами или инструкциями в специальную папку, а потом спрашиваете ответы на вопросы — программа ищет точные места и отвечает.

## Как установить и запустить

1. Скачайте все файлы с GitHub:
```
git clone https://github.com/agentum-ai/RAG-base.git
cd RAG-base
```

2. Установите нужные программы одной командой:
```
pip install -r requirements.txt
```

3. Получите бесплатный ключ на сайте OpenAI (platform.openai.com/api-keys). Создайте файл api_key.txt и напишите в него только ваш ключ, например:
```
sk-proj-abc123...
```

4. Создайте папку knowledge_base рядом с другими файлами (её нет в репозитории — создайте сами).

5. Запустите программу:
```
python rag_pro_final.py
```

Программа проверит файлы в knowledge_base, создаст поиск и скажет: "Готово! Задавайте вопросы".

## Как добавить свои документы

1. В папку knowledge_base положите файлы: Word (DOCX), PDF или простые текстовые (TXT).
2. Запустите конвертацию:
```
python convert_docx.py  # для Word
python convert_pdf.py   # для PDF
```
3. Если файлы на русском открываются с ошибками (кракозябры, CP1251), запустите:
```
python fix_encoding.py
```
4. Если поиск не видит новые файлы, удалите папку faiss_index и запустите программу заново.

## Как задавать вопросы

В окне программы пишите вопросы, например:
- "какая цена контракта"
- "кто директор"
- "пункт 2.3 договора"

Программа ответит с указанием источника. Чтобы выйти, напишите exit.

## Что в файлах

- rag_pro_final.py — главная программа, запускайте её.
- requirements.txt — для установки одной командой.
- convert_docx.py — превращает Word в текст.
- convert_pdf.py — превращает PDF в текст.
- fix_encoding.py — исправляет русские буквы в файлах (CP1251 → UTF-8).
- api_key.txt.example — пример, как создать ключ.

Папки knowledge_base и faiss_index создаются при работе — их нет в скачивании.
